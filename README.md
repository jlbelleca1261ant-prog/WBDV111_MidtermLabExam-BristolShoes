## Short Project Description
A static website for **Bristol Shoes & Shoe Factory - Marikina Branch**, a proudly Filipino shoe brand from Marikina City, the Shoe Capital of the Philippines. The website shows their handcrafted leather shoes for men and women, their story, and store information.

Built using **HTML** and **CSS** for the WBDV111 Midterm Laboratory Examination.

## Features Implemented

### Website Structure (4 Pages)
- **Home Page** (`index.html`) - Hero banner, about preview, feature cards, and CTA
- **About Page** (`about.html`) - Brand story, values, and stats
- **Products Page** (`products.html`) - Men's and Women's shoe collections
- **Contact Page** (`contact.html`) - Contact form, store info, and Google Map

### Design & Functionality
- Responsive design (works on phones, tablets, and desktops)
- Navigation bar with active page highlighting
- Navy Blue + Gold color scheme
- Google Fonts (Playfair Display + Inter)
- Images with alt text for accessibility
- Semantic HTML5 tags (nav, section, footer)
- External CSS stylesheet (no inline styles)
- Proper folder structure (css/, images/)
